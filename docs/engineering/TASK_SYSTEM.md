# 桐宝学习 - 任务系统设计（Task System）

# 1. 系统定位

任务系统是 桐宝学习 的核心执行模块。

目标：

- 将学习内容、探索任务、创造力挑战、AI任务拆分成小任务
- 确保孩子每天都能完成可见成长
- 与成长循环、积分、等级、宠物、世界解锁完全绑定

---

# 2. 任务分类

## 2.0 当前阶段聚焦

当前阶段先聚焦：

```text
二年级语文生字学习、复习、测验与打卡
```

其他年级、数学、英语和更多语文任务类型仅做结构预留。

这一阶段的目标是通过“新学 + 遗忘曲线复习 + 小测验”快速增加孩子识字量。

## 2.1 校内学习任务

- 语文任务
- 数学任务
- 英语任务
- 错题复习任务
- 阅读任务

特点：

- 对应人教版教材
- 年级分层
- 积分奖励固定
- 支持连续成长加成

---

## 2.2 校外探索任务

- AI探索任务
- 动手实验任务
- 创造力任务
- 世界知识探索
- 阅读与写作任务

特点：

- 任务可选
- 高积分奖励
- 解锁宠物/世界/成就
- 激发好奇心和创造力

---

# 3. 年级分层设计

| 年级 | 校内任务类型 | 校外任务类型 |
|------|-------------|-------------|
| 1-2年级 | 拼音、生字、口算、基础阅读 | 简单绘画、基础实验、故事创作 |
| 3-4年级 | 阅读理解、乘除法、写话 | AI探索、科学实验、创意设计 |
| 5-6年级 | 作文、分数、几何、长篇阅读 | AI高级任务、复杂实验、发明设计 |

---

# 4. 任务生成规则

## 4.1 日任务生成

- 每天生成：
    - 校内学习任务 3~5 个
    - 探索任务 1~2 个
- 任务优先按照：
    1. 必须完成的教材核心知识点
    2. 上次错题复习
    3. 连续成长/成就任务
    4. 可选探索任务

---

## 4.2 任务拆解原则

大任务 → 小任务：

- 语文：课文 → 生字 → 拼音 → 阅读理解 → 小作文
- 数学：章节 → 口算 → 应用题 → 趣味挑战
- 英语：单元 → 单词 → 句型 → 跟读 → 阅读理解
- 探索任务：创意项目 → 步骤拆分 → 每步奖励 积分

---

# 5. 任务状态管理

任务状态：

| 状态 | 描述 |
|------|------|
| pending | 待完成 |
| in_progress | 正在进行 |
| submitted | 已提交打卡材料 |
| completed | 已完成 |
| review | 复习任务 |

任务状态变化：

- 完成任务 → 更新 积分 → 更新成长系统 → 更新宠物/世界
- 错题复习 → 更新 积分 → 再次提升掌握度
- 家长评价不阻塞任务完成，只用于追加激励和调整后续复习计划

---

# 6. 积分奖励策略

## 6.1 校内任务

- 基础任务：10~50 积分
- 连续成长加成：+5~20%
- 难度加成：+10~30%（高年级）

## 6.2 校外探索任务

- 基础奖励：30~80 积分
- 创意/挑战任务额外奖励：50~100 积分
- 完成高难度/稀有任务 → 宠物进化/世界解锁/特殊成就

---

# 7. 任务完成流程

1. 孩子点击任务开始
2. 任务内容展示
3. 完成任务并提交打卡材料
4. 系统更新状态：
    - 任务状态 → completed
    - 基础 积分 奖励 → 立即更新等级
    - 宠物成长 → 更新进化
    - 世界进度 → 更新解锁
    - 成就 → 触发新徽章/称号
5. 弹出正反馈动画与鼓励提示
6. 家长端实时同步更新报告
7. 家长稍后评价打卡效果：
    - 很好 → 追加 积分，提升掌握度
    - 还可以 → 记录反馈，正常进入复习计划
    - 再练一次 → 不扣 积分，标记为易忘并优先复习

---

# 8. 任务复习与错题

- 错题自动记录到任务系统
- 生成复习任务 → 积分奖励
- 按间隔重复原则安排复习（1天、3天、7天、14天）
- 复习任务可绑定宠物/世界成长

当前生字任务采用固定间隔复习：

```text
学习当天
第 1 天复习
第 3 天复习
第 7 天复习
第 14 天复习
```

复习优先级：

1. 到期复习字。
2. 家长标记“再练一次”的易忘字。
3. 新学生字。

# 9. 任务优先级

| 类型 | 优先级 | 说明 |
|------|--------|------|
| 校内必做任务 | 高 | 对应教材知识点，必须完成 |
| 错题复习 | 高 | 巩固基础 |
| 连续成长任务 | 中 | 保持粘性 |
| 校外探索任务 | 中 | 激发兴趣与创造力 |
| 可选挑战任务 | 低 | 提供额外成长和趣味 |

---

# 10. 最终目标

任务系统确保：

- 每个任务都与成长循环绑定
- 孩子每次完成任务都有可见成长
- 校内任务保证成绩提升
- 校外任务保证兴趣与未来能力培养
- 家长端能清楚看到孩子进度与成长
