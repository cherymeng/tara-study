# 14. 页面跳转原则

页面跳转必须：

* 简单
* 清晰
* 低认知负担

避免：

* 深层级页面
* 复杂导航

# 17. 最终目标

整个 App Flow 的最终目标：

不是：

# “完成任务”

而是：

# 让孩子形成长期成长习惯。
