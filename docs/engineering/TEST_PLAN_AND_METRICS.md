# 桐宝学习 - 测试计划与核心指标（Test Plan & Metrics）

# 1. 文档目标

本文件用于定义：

* MVP测试策略
* 核心功能测试
* 用户体验验证
* 成长循环验证
* 数据指标体系

目标：

# 验证：

孩子是否真正形成成长反馈与长期使用意愿。

---

# 2. MVP阶段核心验证目标

MVP阶段最重要的：

不是：

* 功能数量

而是：

# 验证成长循环是否成立。

---

# 3. 核心验证问题

必须验证：

## 孩子侧

* 是否理解任务
* 是否愿意完成任务
* 是否对XP反馈有反应
* 是否愿意第二天回来

---

## 家长侧

* 是否理解成长价值
* 是否愿意持续让孩子使用
* 是否认为孩子有变化

---

# 4. 功能测试范围

MVP阶段：

必须测试：

| 模块       | 是否重点测试 |
| -------- | ------ |
| 今日任务系统   | 是      |
| XP系统     | 是      |
| 等级系统     | 是      |
| streak系统 | 是      |
| 宠物系统     | 是      |
| 家长报告     | 是      |
| 动画系统     | 是      |

---

# 5. UI测试

# 核心目标：

验证：

# “孩子是否喜欢打开产品”

---

## 测试内容

包括：

* 页面是否清晰
* 任务是否容易理解
* 动画是否舒适
* 是否存在认知压力

---

## 重点观察

包括：

* 是否迷路
* 是否不会点
* 是否害怕失败
* 是否感觉压力大

---

# 6. 动画测试

动画必须：

* 强化成长感
* 不干扰学习
* 不刺激过强

---

## 测试重点

包括：

* XP动画反馈是否明显
* 升级是否有仪式感
* 宠物是否增强陪伴感
* 世界解锁是否增强探索欲

---

# 7. 任务系统测试

# 重点：

任务是否：

* 容易开始
* 容易完成
* 有成长感

---

## 测试内容

包括：

* 任务时长是否合理
* XP奖励是否合适
* 是否愿意继续完成下一个任务

---

# 8. streak系统测试

# 核心目标：

验证：

# “长期习惯是否形成”

---

## 测试内容

包括：

* 连续成长是否有效
* 是否形成每日打开习惯
* 中断后是否容易放弃

---

# 9. 宠物系统测试

# 核心目标：

验证：

# “情感连接是否成立”

---

## 测试内容

包括：

* 是否喜欢宠物
* 是否愿意互动
* 是否关心宠物成长
* 宠物是否提升使用欲望

---

# 10. 家长端测试

# 核心目标：

验证：

# “家长是否认可成长价值”

---

## 测试内容

包括：

* 是否看懂成长报告
* 是否感受到孩子变化
* 是否愿意鼓励孩子

---

# 11. MVP 核心指标（Metrics）

# 当前阶段最重要指标：

不是：

# DAU

而是：

# “成长循环是否成立”

---

# 12. 核心指标体系

## 12.1 D1留存

# 指标：

第二天是否回来。

目标：

> ≥ 40%

---

## 12.2 D7留存

# 指标：

7天后是否继续使用。

目标：

> ≥ 20%

---

## 12.3 streak形成率

# 指标：

是否形成连续成长。

目标：

> ≥ 30%

---

## 12.4 任务完成率

# 指标：

是否愿意完成任务。

目标：

> ≥ 70%

---

## 12.5 XP反馈有效性

# 指标：

孩子是否关注XP/等级。

观察：

* 是否主动查看成长页
* 是否主动想升级

---

## 12.6 宠物互动率

# 指标：

宠物是否产生情感价值。

观察：

* 是否主动点宠物
* 是否关注宠物变化

---

# 13. 用户行为观察重点

观察：

## 孩子是否：

* 愿意打开产品
* 愿意完成任务
* 愿意继续成长
* 愿意探索

---

## 家长是否：

* 愿意继续让孩子使用
* 愿意鼓励孩子
* 愿意每天查看成长

---

# 14. MVP阶段不要过度关注

当前阶段：

# 不重点关注：

* 收费
* 商业化
* 大规模增长

---

# 当前阶段真正重要：

# 产品价值是否成立。

---

# 15. 数据埋点建议

建议埋点：

| 事件                 | 描述       |
| ------------------ | -------- |
| task_start         | 开始任务     |
| task_complete      | 完成任务     |
| xp_gain            | 获得XP     |
| level_up           | 升级       |
| streak_update      | streak变化 |
| pet_interact       | 宠物互动     |
| world_unlock       | 世界解锁     |
| parent_view_report | 家长查看报告   |

---

# 16. 测试阶段建议

# 第一阶段：

小规模真实儿童测试。

建议：

* 5~20个孩子
* 真实家庭环境
* 连续观察1~2周

---

# 17. 当前最重要的验证

当前阶段：

# 最重要验证：

孩子是否会感觉：

> “我在成长。”

如果这个成立：

产品就有长期价值。

---

# 18. 最终目标

通过测试与数据：

持续优化：

* 成长循环
* 正反馈
* 情感连接
* 长期习惯

最终打造：

# 真正让孩子愿意长期成长的产品。
