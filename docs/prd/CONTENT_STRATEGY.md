# 桐宝学习 - 内容战略系统（Content Strategy）

# 1. 系统定位

Content Strategy 是 桐宝学习 的长期内容体系。

目标：

* 支撑长期成长
* 保证学习效果
* 保持孩子兴趣
* 提供长期探索内容
* 建立内容护城河

---

# 2. 内容系统核心理念

桐宝学习 的内容：

不是：

* 题库堆积
* 机械刷题
* 大量重复练习

而是：

# 成长驱动内容系统。

所有内容都应该：

* 帮助成长
* 激发兴趣
* 提供探索感
* 建立长期习惯

---

# 3. 内容结构

产品内容分为：

## 3.1 校内同步内容（80%）

包括：

* 人教版同步
* 学科任务
* 错题复习
* 阅读理解
* 写作训练

目标：

# 提升成绩。

---

## 3.2 探索成长内容（20%）

包括：

* AI探索
* 科学实验
* 世界知识
* 创造任务
* 阅读拓展

目标：

# 培养未来能力。
