# Tara Study - AI探索成长系统（AI Exploration System）

# 1. 系统定位

AI探索成长系统是Tara Study差异化核心模块，旨在培养：

- 好奇心
- 创造力
- AI时代能力
- 探索欲
- 实践能力

系统与校内学习同步，但更偏向：

- 校外能力拓展
- 动手实验
- 创造力训练
- AI素养启蒙

---

# 2. 系统目标

- 在保证校内成绩提升的同时，培养未来能力
- 让孩子在探索中获得成长值（XP）
- 与成长循环、宠物成长、世界解锁、成就系统完全绑定
- 提供长期兴趣驱动，形成持续学习动力

---

# 3. AI任务体系

## 3.1 AI任务类型

1. **创意生成**
    - AI绘画（如画恐龙、设计机器人）
    - AI故事生成（短篇故事、童话）
    - AI作文辅助（提供写作灵感）
2. **AI互动**
    - AI问答探索（科学知识、历史知识）
    - AI实验指导（步骤提示、结果分析）
3. **AI学习辅助**
    - 个性化学习建议
    - 学科知识讲解
    - 阅读理解辅助

## 3.2 AI任务奖励

- 完成任务 → 获得 XP
- 高难度/创意任务 → 宠物成长、世界解锁
- 任务可重复或改造 → 提高长期粘性

---

# 4. 动手实验与创造力

## 4.1 动手实验类型

- 科学实验（火山喷发、小电路、植物观察）
- DIY项目（小手工、创意装置）
- 家庭探索（自然观察、实验记录）

## 4.2 创造力任务

- 绘画（主题绘画、想象星球）
- 写作（故事创作、小短文）
- 发明设计（未来机器人、动物设计）

## 4.3 任务与成长体系绑定

- 每个任务完成 → 获得 XP
- 任务可解锁宠物新技能、新世界元素、新成就徽章

---

# 5. 世界探索

## 5.1 世界主题

- 森林世界（自然探索、动物知识）
- 科学世界（实验、科学原理）
- AI未来世界（AI实验、创造力任务）
- 海洋世界（海洋生物、环保探索）
- 太空世界（宇宙、天文探索）

## 5.2 解锁机制

- 学校任务完成 → 基础世界开放
- AI探索/创造力任务完成 → 高级世界解锁
- 连续成长/高难度任务 → 稀有资源或特殊场景解锁

---

# 6. 学习与探索结合

- AI任务 + 校内任务 = XP叠加
- 动手实验 + 宠物成长/世界解锁
- 阅读 + 创造力任务 = 成长曲线提升

> 核心理念：探索成长与校内学习融合，而不是割裂

---

# 7. AI陪伴系统

## 7.1 系统定位

- AI角色既是学习辅助，也是成长伙伴
- AI提供温暖鼓励、正反馈、探索建议
- 孩子成长 → AI互动进化 → 宠物/世界反馈同步

## 7.2 AI角色示例

- 小机器人（科技引导）
- 小精灵（创意辅助）
- 小龙（探索伙伴）
- 可升级/可成长/可进化

## 7.3 AI互动原则

- 温暖、鼓励、正向
- 不惩罚、不比较、不施压
- 提供创造与探索的自由空间

---

# 8. 任务激励体系

1. XP成长值
2. 宠物成长与技能解锁
3. 世界场景解锁
4. 成就徽章
5. 连续成长奖励

> 目的：让孩子在完成探索任务时获得与校内学习同样的成就感

---

# 9. 系统设计原则

- 与校内学习系统完全兼容，形成统一成长循环
- 任务必须趣味、可操作、可见成长
- 提供多样化探索任务，满足不同兴趣
- 逐步增加挑战，增强长期粘性
- AI辅助保证孩子安全、正向成长

---

# 10. 最终目标

- 孩子能同时获得：
    - 学科能力提升
    - 创造力与实践能力
    - AI时代素养
    - 自律习惯
    - 长期成长动力

- 家长能：
    - 精准掌握孩子能力
    - 激励孩子探索兴趣
    - 形成正向陪伴环境
