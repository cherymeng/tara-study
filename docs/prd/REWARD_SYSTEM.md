# 桐宝学习 - 奖励与正反馈系统（Reward System）

# 1. 系统定位

Reward System 是 桐宝学习 的成长反馈系统。

目标：

* 建立长期学习动力
* 强化成长感
* 增强坚持感
* 提供即时正反馈
* 形成长期自驱力

核心原则：

# 奖励不是目的。

# 成长感才是目的。

---

# 2. 奖励系统核心理念

传统学习：

* 延迟反馈
* 只有考试结果
* 缺少成长体验

桐宝学习：

# 每一次成长，

都能被看见。

---

# 3. 奖励系统结构

奖励系统包括：

* XP成长值
* 等级成长
* 连续成长奖励
* 成就系统
* 宠物成长
* 世界解锁
* 稀有奖励
* AI鼓励
* 家长鼓励

所有奖励：

统一进入：

# 成长循环。

---

# 4. XP 奖励系统

XP 是最核心奖励。

---

## 4.1 XP 来源

包括：

* 学习任务
* 阅读

* 科学实验

---

包括：

* 数值动画

* 音效
* 宠物互动
* 成长提示

目标：

让孩子：

# 感受到成长。

* 成长

让孩子逐渐形成：

* 自驱学习
* 长期坚持
* 成长心态
* 自信心

最终让孩子感觉：

> “成长本身，就是一件开心的事情。”
