# 桐宝学习 - 家长协同系统（Parent System）

# 1. 系统定位

Parent System 是 桐宝学习 的家庭协同成长系统。

目标：

* 帮助家长理解孩子成长
* 降低教育焦虑
* 提供科学陪伴方式
* 建立长期家庭成长机制

核心理念：

# 家长不是监督者，

而是成长陪伴者。

---

# 2. 系统目标

Parent System 的核心目标：

## 2.1 让家长“看见成长”

很多家长只能看到：

* 分数
* 作业
* 考试结果

但看不到：

* 习惯变化
* 学习稳定性
* 长期成长趋势

桐宝学习 需要：

# 把成长可视化。

---

## 2.2 降低家庭冲突

传统学习过程：

容易出现：

* 催促
* 批评
* 对抗
* 焦虑

产品需要：

# 让“监督”变成“陪伴”。

---

## 2.3 提供科学教育辅助

帮助家长：

* 理解孩子成长阶段
* 理解学习状态
* 获得鼓励建议
* 避免错误教育方式

---

# 3. 家长端核心功能

---

## 每月成长报告

---

# 3.3 成长可视化系统

# 产品必须让家长：

“看见成长”。

---

## 成长展示内容

包括：

* 积分成长曲线
* 等级成长
* 连续成长记录
* 宠物成长
* 世界解锁进度
* 成就徽章

---

## 核心目标

让家长感觉：

> “孩子真的在长期成长。”

而不是：

> “今天只是完成了作业。”

---

# 3.4 家长鼓励系统

家长可以：

* 点赞
* 发送鼓励
* 解锁奖励
* 设定成长目标

当前二年级生字阶段，家长评价规则如下：

* 孩子提交打卡后，任务已完成，并立即获得基础 积分。
* 家长评价不阻塞孩子完成任务，也不阻塞基础 积分。
* 家长评价为“很好”时，可以追加 积分。
* 家长评价为“再练一次”时，不扣除 积分，只把相关生字放入后续复习队列。
* 家长反馈文案应以鼓励和下一步建议为主，避免失败感。

---

## 鼓励形式

例如：

* “今天进步很大！”
* “连续坚持太棒了！”
* “你越来越厉害了！”

---

## 系统原则

强化：

* 鼓励
* 认可
* 成长感

避免：

* 批评
* 对比
* 施压

---
