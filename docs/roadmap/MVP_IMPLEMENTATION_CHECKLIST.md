# MVP 实施清单

当前版本：

```text
MVP V1
```

目标：

验证儿童成长循环是否成立。

---

# MVP 核心目标

构建如下循环：

```text
每日任务
    ↓
完成任务
    ↓
获得成长值
    ↓
宠物成长
    ↓
解锁世界内容
    ↓
获得成就感
    ↓
继续完成任务
```

---

# MVP 范围

## 包含

- 二年级语文生字每日任务
- 生字打卡
- 遗忘曲线复习
- 生字小测验
- 成长值
- 等级系统
- 宠物成长
- 世界地图
- 成就系统
- 家长查看端

## 不包含

- 其他年级
- 数学、英语等其他学科
- 大题库
- 真实拍照/录音上传
- AI批改
- AI聊天
- AI老师
- AI陪伴
- 内容生成
- 支付系统
- 会员系统
- 社交系统

- [ ] 录音模拟上传

---

## 任务状态

- [ ] 未开始
- [ ] 进行中
- [ ] 已提交
- [ ] 已完成

规则：

- [ ] 孩子提交后立即完成任务并获得基础 XP
- [ ] 家长评价不阻塞完成
- [ ] 家长评价很好可追加 XP
- [ ] 家长评价再练一次只影响复习计划，不扣 XP
